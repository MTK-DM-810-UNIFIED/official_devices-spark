# Clean flash:
- Unlock bootloader 
- Flash boot and vendor_boot
- Download ROM
- Reboot to recovery
- click on apply update 
- adb sideload the zip 
- Format data
- Reboot and voila!

# Updating to a newer build (dirty flash):
- Update via OTA Updater, or
- Reboot to recovery sideload 
- Reboot and voila!
